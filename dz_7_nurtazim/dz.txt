Задание 1

Пользователь вводит название страны в поле, и нажимает кнопку "Найти".
Выведите название этой страны, затем найдите все страны, с которыми граничит эта страна и выведите их полное название в интерфейс.

Пример:
Страна: Kyrgyzstan
Граничит с:
●       China
●       Kazakhstan
●       Tajikistan
●       Uzbekistan


●       Использовать Promise.all
●   	Выводить информацию только после того как все запросы будут закончены
●   	Можно добавить прелоадер, чтобы выглядело красиво.
●   	Для поиска страны по названию API:
○   	https://restcountries.com/v3.1/name/{name}
○   	где вместо Name подставляете название, например Kyrgyzstan
●   	Для поиска страны по коду:
○   	https://restcountries.com/v3.1/alpha/{code}
○   	где вместо {code} подставляете код, например KGZ
